## Огляд

Огляд
Модуль дозволяє отримувати опис доступних інструментів або ініціювати виконання завдань шляхом звернення до зовнішніх API за адресою <http://127.0.0.1:8000/v1>. Він відповідає за запуск серверної логіки, яка використовує параметри, визначені у файлі config.json, для взаємодії з зовнішніми системами.

## Поведінка

1. При отриманні команди `list` виводиться список доступних інструментів.
2. При отриманні команди `schema` виводиться маніфест усіх доступних інструментів.
3. При отриманні команди `agent` відбувається звернення до мережі за параметрами, які визначені в змінних середовища (таких як `OMLX_BASE_URL` за замовчуванням `http://127.0.0.1:8000/v1` та `OMLX_MODEL`), для виконання завдання за наданим промптом.
4. При отриманні команди `mcp` запускається сервер, який забезпечує комунікацію з зовнішніми оркестраторами. Цей сервер використовує конфігурацію, що зберігається в `config.json`, для визначення шляхів до проектів. Він також ініціалізує механізм журналу для запитів.
5. При отриманні будь-якої іншої команди виконується диспетчеризація, що вимагає коректного JSON-форматування вхідних даних.
6. Результат виконання операції виводиться у стандартний вивід у форматі JSON.
7. Якщо вхідні дані є некоректним JSON, виводиться помилка у стандартний потік помилок.

## Гарантії поведінки
