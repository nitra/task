## Огляд

Файл є централізованим репозиторієм для всіх інструментів, доступних через зовнішні інтерфейси. Він забезпечує доступ до реєстру інструментів, що визначається конфігурацією з файлу .mt.json. Публічні функції, такі як TOOLS та getTool, дозволяють знаходити та отримувати обраний інструмент.

## Поведінка

TOOLS — Масив, що є єдиним джерелом істини для визначення всіх доступних інструментів, які можуть бути викликані інтерфейсом користувача, скриптами оркестратора або великими мовними моделями.
getTool — Знаходить і повертає визначення інструмента за його назвою, або `null`, якщо інструмент не знайдено.

## Публічний API

- TOOLS — Набір доступних інструментів, які може використовувати код.
- getTool — Шукає конкретний інструмент за його назвою.
