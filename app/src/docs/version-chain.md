## Огляд

Визначає зовнішній вигляд артефакту через функцію `artifactConfig` залежно від його типу та отриманого результату. Також структурує послідовність артефактів, групуючи їх у секції для представлення на часовій шкалі, використовуючи `chainSections`.

## Поведінка

artifactConfig визначає візуальну конфігурацію артефакту на основі його типу та результату.
chainSections розбиває послідовність артефактів на секції для часової шкали.

## Публічний API

artifactConfig — Налаштування візуального вигляду артефакту, де результат перезаписує базовий колір (червоний для невдач, бірюзовий для розділених, спавнені).
chainSections — Групує послідовність артефактів, створену бекендом, у секції часової шкали: `mission` (завдання.md) → `attempt NNN` → `terminal` (невирішений/звіт виконання).
