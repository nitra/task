## Огляд

Файл забезпечує аналіз текстового блоку, який містить чернетки виконання. Через функцію `parseRunDraft` він відокремлює завершені кроки, виявлені перешкоди та наступні заплановані спроби. Результатом є об'єкт, що містить окремі рядки для кожної з цих секцій.

## Поведінка

1. Викликати `parseRunDraft` для аналізу тексту, що містить чернетки виконання (як у живій стрічці GUI).
2. Функція витягує секції `Completed`, `Blockers` та `Next Attempt` із наданого тексту.
3. Повертається об'єкт, що містить з'єднані рядки для кожної з витягнутих секцій, або `null`, якщо секція не знайдена чи пуста.
