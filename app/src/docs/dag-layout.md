## Поведінка

1. Викликається `computeDagLayout` для обробки структури воркспейсу.
2. Функція `computeDagLayout` спочатку згладжує вкладену структуру вузлів у плаский список, збираючи всі вузли.
3. Потім для кожного вузла обчислюється його рівень глибини в залежностях. Якщо залежність не знайдена у наданому наборі вузлів, рівень вважається 0. У разі виявлення циклічної залежності, відповідний вузол отримує рівень 0 для запобігання зависанню.
4. Вузли групуються відповідно до обчислених рівнів глибини. У межах одного рівня вузли сортуються за їхнім шляхом.
5. Кожному вузлу присвоюються координати (x, y) на основі його рівня (шару) та позиції у групі вузлів цього рівня.
6. Формуються списки ребер, що відображають прямі залежності між вузлами.
7. На виході надається об'єкт, що містить позиціоновані вузли, список ребер, а також загальну ширину та висоту візуалізації.

## Публічний API

computeDagLayout — Створює розташування елементів для візуалізації графа залежностей (DAG), визначаючи позиції вузлів за рівнем залежності та з'єднуючи їх ребрами.
