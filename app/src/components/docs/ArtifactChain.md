## Огляд

Компонент візуалізує послідовність артефактів, згрупованих за секціями. Він відображає статус кожного артефакту, зокрема його результат або виконавця.

## Поведінка

1. Компонент отримує масив артефактів (`artifacts`) та поточний вибраний артефакт (`selected`) як пропси.
2. На основі отриманих артефактів компонент динамічно генерує секції за допомогою функції `chainSections` з `../version-chain.js`.
3. Для кожної секції відображається заголовок секції.
4. Для кожного артефакту в секції відображається іконка, мітка та метаінформація.
5. При кліку на артефакт компонент випромінює подію `select`, передаючи дані клікнутого артефакту.
6. Артефакт, що відповідає значенню `selected`, підсвічується як активний.
7. Якщо масив артефактів порожній, відображається повідомлення "no artifacts".

## Гарантії поведінки

- Відображення структурованої послідовності артефактів, розділених логічними секціями.
- Надання можливості кліку для вибору конкретного артефакту з масиву.
- Відображення метаданих про артефакт (результат або виконавець) відповідно до його стану.
