Компонент відображає діаграму залежностей (Dependency Graph) у модальному діалоговому вікні. Він візуалізує взаємозв'язки між вузлами (компонентами) у певному робочому просторі.

## Поведінка

1. При отриманні вхідних даних `nodes` (списку вузлів), компонент розраховує макет діаграми залежностей, використовуючи `computeDagLayout`.
2. Вузол `DagViewDialog` відкривається та закривається відповідно до стану, переданого через `modelValue`.
3. Якщо список вузлів порожній, відображається повідомлення про відсутність вузлів.
4. У разі наявності вузлів, відображається векторна графічна діаграма (SVG).
5. Кожен вузол відображається як прямокутник із підписом, його позиція визначається розрахованим макетом.
6. Кожен вузол є інтерактивним; при кліку на вузол компонент емітує подію `select`, передаючи унікальний ідентифікатор (path) цього вузла.
7. Між вузлами відображаються стрілкові лінії (edgePaths), що вказують напрямок залежностей, від вузла-джерела до вузла-цілі.

## Гарантії поведінки

- Компонент гарантує, що діаграма відображається з коректним позиціонуванням вузлів та з'єднань залежно від вхідних даних `nodes`.
- При кліку на вузол завжди генерується подія `select` з коректним ідентифікатором.
- Якщо вхідні дані `nodes` відсутні, компонент коректно відображає стан "порожнього" графа.
