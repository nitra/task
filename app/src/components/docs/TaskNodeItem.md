Компонент відображає один вузол завдання у вигляді рядка, рекурсивно відображаючи дочірні вузли. Він забезпечує інтерактивність для вибору вузла та розкриття/згортання композитних структур.

## Поведінка

1. При натисканні на рядок завдання виконується еміт `select` з об'єктом вузла.
2. Якщо вузол композитний, натискання на іконку розширення перемикає його видимість.
3. При відкритті композитного вузла, компонент рекурсивно рендерить `TaskNodeItem` для кожного його дочірнього елемента, збільшуючи глибину відступу.
4. При натисканні на посилання залежності (`dep`), виконується еміт `select-dep` з елементом залежності.
5. Вузол, що має стан `unresolvable`, візуально закреслюється.
6. Відображаються ідентифікатор вузла, його поточний стан (у вигляді мітки), іконка, ідентифікатор виконавця (за наявності), список залежностей та бюджет часу.

## Гарантії поведінки

- Компонент коректно відображає ієрархію завдань, використовуючи параметр `depth` для визначення відступу.
- Інтерактивність (вибір та розкриття) реалізована через еміти, що дозволяє батьківському компоненту керувати станом програми.
- Відображення візуально відображає мета-дані вузла (стан, залежності, бюджет) відповідно до логіки `stateConfig`.
