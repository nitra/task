## Огляд

Конфігурує середовище збірки та розробки застосунку через Vite. Він ініціалізує Vue та Vue Router, застосовує Vue макроси, впроваджує систему компонування макетів та налаштовує плагін Quasar. Також забезпечує роботу сервера розробки з портом 1420, підтримує Hot Module Replacement (HMR) та дозволяє налаштування URL відповідно до <https://vite.dev/config/>.

## Поведінка

Організовує процес збірки та розробки застосунку, конфігуруючи середовище за допомогою Vite.

1. Ініціалізує плагіни для автоматичного імпорту функцій Vue та Vue Router.
2. Застосовує макроси Vue для забезпечення сумісності шаблонів з утилітами Vue.
3. Інтегрує систему компонування макетів (Layouts).
4. Конфігурує плагін Quasar, вказуючи шлях до змінних Sass.
5. Налаштовує сервер розробки з портом 1420, дозволяє вказати хост, ініціює Hot Module Replacement (HMR) при необхідності, та ігнорує файли, пов'язані з Tauri.
6. Підтримує налаштування URL відповідно до документації <https://vite.dev/config/>.
