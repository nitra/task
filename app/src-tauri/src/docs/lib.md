## Огляд

Файл надає інструменти для роботи з завданнями. Функції включають `scan_tasks`, `find_tasks_dir`, `find_all_tasks_dirs`, `read_task` та `run`.

## Поведінка

scan_tasks
Сканує директорію завдань

find_tasks_dir
Знаходить директорію завдань

find_all_tasks_dirs
Знаходить усі директорії робочих просторів

read_task
Читає файл завдання

run
Запускає застосунок

## Публічний API

scan_tasks — переглядає завдання у файлі.

find_tasks_dir — знаходить директорію завдань.

find_all_tasks_dirs — знаходить усі директорії завдань.

read_task — читає одне завдання.

run — виконує завдання.

## Гарантії поведінки

- Read-only: файл не виконує операцій запису у файлову систему.
- Перехоплює помилки і не пропускає винятків назовні (fail-safe).
- За невдалої перевірки повертає `false`/`null` замість винятку.
- Не звертається до мережі.
