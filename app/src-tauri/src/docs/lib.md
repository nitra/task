# lib.rs

Модуль надає функції для сканування, пошуку та читання файлів завдань у визначених директоріях. Функції забезпечують взаємодію з системними файловими операціями для роботи з структурою завдань.

## Поведінка
1. `scan_tasks -> Result<Vec<TaskNode>, String>`
    * Приймає шлях до директорії.
    * Викликає `mt_scanner::scan_tasks` для сканування завдань у вказаній директорії.
    * Повертає вектор `TaskNode` або помилку у вигляді рядка.

2. `find_tasks_dir -> Result<String, String>`
    * Повертає шлях до директорії завдань.
    * Викликає `mt_scanner::find_tasks_dir`.
    * Повертає рядок шляху або помилку.

3. `find_all_tasks_dirs -> Result<Vec<WorkspaceInfo>, String>`
    * Повертає список об'єктів `WorkspaceInfo`.
    * Викликає `mt_scanner::find_all_tasks_dirs`.
    * Повертає вектор `WorkspaceInfo` або помилку у вигляді рядка.

4. `read_task -> Result<String, String>`
    * Складає повний шлях до файлу `task.md` у вказаній директорії.
    * Зчитує вміст файлу.
    * Повертає вміст файлу або помилку у вигляді рядка.

## Гарантії поведінки
* Функції, що повертають `Result`, повертають `Ok` з результатом або `Err` з описом помилки у вигляді рядка.
* Операції з файловими операціями можуть завершитися невдачею (наприклад, неіснуючий файл або помилка доступу), що призводить до повернення `Err`.
* Помилки, пов'язані з файловими операціями, перетворюються у текстові повідомлення.
