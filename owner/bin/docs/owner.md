## Огляд

Цей компонент обробляє запити, що містять JSON-пейлоад, для взаємодії з інструментами, виводячи каталоги або схеми в JSON. При отриманні команди на виконання інструменту він парсить вхідні дані, ініціюючи роботу `mt-scanner` на основі інформації з `config.json`. У разі помилки парсингу або некоректного завершення роботи `mt-scanner`, компонент повертає відповідний код помилки.

## Поведінка

1. При отриманні команди "list" або "list", програма виводить JSON-об'єкт, що містить повний каталог доступних інструментів.
2. При отриманні команди "schema", програма виводить JSON-об'єкт, що описує схему всіх інструментів.
3. Якщо отримана команда не є "list" чи "schema" та містить JSON-payload, програма намагається розпарсити цей payload як вхідні дані для виконання інструмента. Якщо парсинг не вдався, виводиться помилка.
4. Якщо команда є валідною, програма використовує механізм диспатча для виконання відповідного інструменту за допомогою зовнішньої утиліти `mt-scanner`.
5. Виконання інструменту відбувається з використанням шляхів до конфігурацій: `config.json` в директоріях, визначених у системних конфігураційних файлах додатків (`com.nitra.owner`, `com.nitra.task`), а потім у `/home/user/www`. Якщо жоден з цих джерел не знайдений, пошук завершується.
6. Якщо інструмент не має визначеного CLI-шляху, ініційована команда вважається внутрішньою (in-app only) і виконання неможливе.
7. Якщо `mt-scanner` не знайдено (ні через змінні середовища, ні через заздалегідь визначені шляхи), виникає помилка.
8. Якщо команда вимагає виконання, `mt-scanner` виконується з JSON-payload як аргументом. У разі команди "workspaces" виконується `mt-scanner` з аргументами "workspaces" та списками кореневих папок, отриманих з конфігурацій.
9. Результат виконання `mt-scanner` зчитується з stdout; якщо статус процесу не нульовий, виводиться помилка з stderr.
10. Розпарсений вивід (JSON) виводиться на stdout як результат виконання. Якщо вхідний JSON є некоректним або виконання завершується з помилкою, відповідний код виходу (2 або 1) передається системі.
