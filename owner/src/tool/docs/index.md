## Огляд

Надає публічну функцію `dispatch` як єдину точку входу для прямих викликів tool-поверхні. Приймає запит на виклик інструмента й передає його на виконання доступному механізму tool-поверхні, не змінюючи файлову систему чи базу даних самостійно.

## Поведінка

1. `dispatch` надає єдину точку входу в tool-поверхню для прямих викликів із UI.

2. Приймає запит від обробників подій інтерфейсу, щоб вони не містили власної бізнес-логіки виклику інструментів.

3. Передає запит у доступний транспорт застосунку для виконання відповідної дії.

4. Не змінює дані у файловій системі чи базі даних самостійно; відповідає лише за делегування виклику.
