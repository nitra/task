## Огляд

Файл створює точку входу для Vue-інтерфейсу й прив’язує відображення до кореневого вузла `#app`. Він потрібен, щоб застосунок стартував у стандартному місці монтування та використовував Quasar як спільну UI-основу.

## Поведінка

1. Ініціалізується клієнтський Vue-додаток і готується базова оболонка інтерфейсу.
2. Підключається Quasar як UI-основа з глобальними налаштуваннями: сповіщення доступні в усьому застосунку, тема підлаштовується під системний режим.
3. Застосовується набір Material Symbols для єдиного стилю іконок у всіх екранах.
4. Завантажуються глобальні стилі Quasar, щоб інтерфейс мав узгоджений вигляд.
5. Додаток монтується в кореневий вузол сторінки `#app` і стає видимим для користувача.
