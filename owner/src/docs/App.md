Цей компонент задає базовий каркас сторінки для застосунку owner: верхній брендований header і основну область вмісту. Він не містить бізнес-логіки, а лише формує постійний інтерфейс оболонки.

## Поведінка

1. Відображає layout сторінки з фіксованою структурою header + content.
2. Показує у верхній панелі бренд owner з декоративною точкою.
3. Розміщує основний екран застосунку всередині центральної області сторінки.
4. Підлаштовує вигляд header під темну або світлу тему.
5. Забезпечує компактну, стабільну навігаційну шапку без зайвих елементів.

## Гарантії поведінки

- Верхня панель завжди присутня на сторінці.
- Основний вміст завжди рендериться в окремій page-області.
- Візуальне оформлення header узгоджується з темою інтерфейсу.
- Компонент не змінює стан застосунку і не виконує побічних дій.
