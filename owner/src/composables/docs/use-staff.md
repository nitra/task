## Огляд

Файл забезпечує можливість викликати механізм взаємодії зі штабом LLM через публічну функцію `useStaff`. Він дозволяє ініціалізувати це підключення та отримувати результати взаємодії у вигляді детального огляду прийнятого рішення та контраргументів, висунутих проти цього рішення.

## Поведінка

1. Викликається `useStaff` для ініціалізації механізму взаємодії зі штабом LLM.
2. Після ініціалізації можна викликати `requestBriefing` для отримання детального огляду рішення.
3. Після ініціалізації можна викликати `requestObjection` для отримання контраргументів проти рішення.

## Публічний API

useStaff — створює резюме та найбільш вагомий контраргумент для прийняття певного рішення.
