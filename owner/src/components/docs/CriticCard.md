Компонент показує один вердикт критика у вигляді картки з контекстом вузла, правилом, робочою областю та текстом знахідки. Він призначений для перегляду заперечення й явного його відхилення власником.

## Поведінка

1. Відображає заголовок вердикту з іконкою, шляхом вузла, міткою правила та назвою workspace.
2. Підсвічує вердикт кольором залежно від його ваги: критичні випадки, попередження або інформаційний рівень.
3. Показує текст знахідки окремим блоком під заголовком.
4. Дає змогу відхилити заперечення через кнопку дії.
5. Після відхилення повідомляє батьківський вміст подією dismiss.

## Гарантії поведінки

- Заперечення завжди показується як окремий, помітний елемент для ручного підтвердження.
- Колірна індикація стабільно відображає серйозність вердикту.
- Кнопка дії не змінює вердикт сама по собі, а лише сигналізує про намір його відхилити.
- Вміст картки лишається прив’язаним до переданого вердикту без прихованих перетворень.
