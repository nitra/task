Компонент показує карту воркспейсів як набір карток із коротким зведенням станів у кожному з них. Він призначений для швидкого огляду поточного розподілу станів без будь-яких дій із даними.

## Поведінка

1. Для кожного воркспейсу відображає окрему картку з назвою та шляхом.
2. Для кожного воркспейсу збирає підсумок станів усіх вузлів у його дереві.
3. Показує по одному chip на кожен стан із кількістю вузлів цього стану.
4. Якщо в межах воркспейсу немає жодного стану для показу, виводить повідомлення про порожній граф.
5. Якщо воркспейсів немає взагалі, показує окреме повідомлення про відсутність воркспейсів.
6. Обхід дерева враховує вкладені вузли, тож зведення відображає весь піддеревний вміст.

## Гарантії поведінки

- Компонент не змінює дані й не запускає жодних побічних дій.
- Підсумок станів завжди формується лише для видимих воркспейсів.
- Порожні або відсутні гілки дерева не ламають відображення зведення.
- Коли немає даних для карти, користувач отримує явне повідомлення замість порожнього екрана.
