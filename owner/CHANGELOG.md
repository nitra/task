# Changelog

## [0.3.1] - 2026-07-12

### Changed

- release: app@0.5.4, owner@0.2.1

## [0.3.0] - 2026-07-12

### Added

- M2 — критик (перша ітерація): детермінований прогін після кожного rescan (мертві/неіснуючі залежності, цикли взаємного очікування, прострочені дедлайни, гілки без прогресу >7 днів включно з непризначеними) + семантичний LLM-прогін на вимогу (дублювання, суперечності цілей; по одному виклику omlx на воркспейс). Вердикти — картки у черзі рішень з явним «відхилити заперечення»; межа ретро-принципу: критик аналізує граф, не людей.

## [0.2.1] - 2026-07-11

### Changed

- 🐛 fix(app): codesign релізу — journal назад у src/bin, docs-дзеркало у .gitignore

## [0.2.0] - 2026-07-11

### Added

- M1 — LLM-плановик: діалог «Нова ціль» (інтент → 2 альтернативні декомпозиції різними лінзами через omlx) з вибором плану; tools `create_goal`/`draft_plan` пишуть вузол цілі та immutable `plan_NNN.md` (children валідуються mt-core до запису) — далі план проходить штатний plan-review гейт у черзі рішень.

## [0.1.0] - 2026-07-11

### Added

- M0 owner-застосунку (спека `docs/specs/260711-owner-app.md`): адаптивний перший екран Рішення → Бриф → Карта, черга рішень з лісу mt-воркспейсів (approve/reject плану, прийняття роботи), дельта «що змінилось» + особисті задачі, tool-поверхня з headless-входом `bin/owner.mjs`, Rust-бекенд над mt-core.
