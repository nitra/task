## ADR Редизайн UI до простого, лаконічного, технологічного стилю

## Context and Problem Statement
Існуючий UI Quasar + Vue Tauri-додатку мав важкий яскраво-синій elevated заголовок і неоднорідні індикатори стану, що суперечило вимозі «простого, лаконічного, технологічного та збалансованого» дизайну. Користувач ініціював повний редизайн візуального стилю.

## Considered Options
* Редизайн з плоским заголовком, єдиним `state-config.js` та уніфікованими state-pill компонентами
* Інші варіанти в transcript не обговорювалися.

## Decision Outcome
Chosen option: "Редизайн з плоским заголовком, єдиним `state-config.js` та уніфікованими state-pill компонентами", because користувач задав напрямок «простий, лаконічний, технологічний, збалансований», а реалізація вилучила elevated заголовок на користь плоского із hairline-роздільником, централізувала палітру станів у `state-config.js` і уніфікувала відображення стану в обох компонентах.

### Consequences
* Good, because єдине джерело істини `state-config.js` гарантує консистентність кольорів і іконок станів між `TaskGraph.vue` та `TaskNodeItem.vue`.
* Good, because transcript фіксує очікувану користь: обидва теми (dark і light) відрендерились без артефактів у headless Chrome.
* Bad, because transcript не містить підтверджених негативних наслідків.
