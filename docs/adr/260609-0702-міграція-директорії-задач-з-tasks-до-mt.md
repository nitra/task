* Інші варіанти в transcript не обговорювалися.

## Decision Outcome

## ADR Виокремлення сканера у самодостатній Rust crate `mt-scanner`

## Context and Problem Statement
Вся логіка сканування задач (`TaskState`, `TaskNode`, `WorkspaceInfo`, `scan_tasks`, `find_all_tasks_dirs` та допоміжні функції) знаходилася безпосередньо у `app/src-tauri/src/lib.rs` разом із Tauri-специфічними `#[tauri::command]` обгортками. Паралельно у `@7n/mt` існував окремий JS-сканер (`lib/core/scanner.mjs`). Користувач хотів єдину реалізацію сканера, яку можна підключати і до Tauri-додатку, і до npm-пакету `@7n/mt`.

## Considered Options
* Окремий Rust crate `mt-scanner` (lib + binary), Tauri підключає як Cargo-залежність, npm викликає бінарник через subprocess
* Залишити логіку у `lib.rs` Tauri-додатку
* Зберегти JS-реалізацію `scanner.mjs`
* Інші варіанти в transcript не обговорювалися.

## Decision Outcome
Chosen option: "Окремий Rust crate `mt-scanner` (lib + binary)", because "як сканер який ми написали на rust зробити самодостатнім і щоб він був частиною проекту @7n/mt (замість варіанту на js) але також ми собі його могли підключити до свого проекту" — вимога мати єдину реалізацію, придатну для reuse без копіювання коду.

### Consequences
* Good, because `mt/scanner/src/lib.rs` містить увесь scanner-код з pub API; `mt/scanner/src/main.rs` реалізує CLI: `mt-scanner scan <tasks_dir>` та `mt-scanner workspaces [<start_dir>]` → JSON stdout; `app/src-tauri/src/lib.rs` скорочено з ~498 до ~44 рядків (лише Tauri-обгортки через `mt_scanner::*`); обидва crate успішно білдяться (`cargo build`); CLI підтверджено: `mt-scanner workspaces` та `mt-scanner scan` повертають коректний JSON.
* Bad, because path-залежність (`mt-scanner = { path = "../../../mt/scanner" }`) прив'язує Tauri-додаток до відносного шляху у файловій системі; на момент завершення сесії обговорювалися варіанти усунення (crates.io publish, git-залежність, Cargo workspace) — рішення не прийнято.

## More Information
Нові файли: `mt/scanner/Cargo.toml`, `mt/scanner/src/lib.rs`, `mt/scanner/src/main.rs`. Оновлено: `app/src-tauri/Cargo.toml` (додано `mt-scanner = { path = "../../../mt/scanner" }`), `app/src-tauri/src/lib.rs` (thin Tauri wrappers). Публікація на crates.io або git-залежність як наступний крок для усунення path-залежності.
